Clockwise is a lightweight, low-level systems programming language inspired by C. The language focuses on:

- Predictability and performance: small runtime and straightforward semantics.
- Readability: C-like syntax with clear conventions.
- Interop: easily call into C libraries and OS APIs.

This folder contains developer-facing documentation: language overview, specification, usage, and contribution guidance.

See LANGUAGE_SPEC.md for the language surface and USAGE.md for getting started.
